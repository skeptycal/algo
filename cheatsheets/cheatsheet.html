Algorithms and Data Structures Cheatsheet


We summarize the performance characteristics of classic algorithms and data structures for sorting, priority queues, symbol tables, and graph processing.

We also summarize some of the mathematics useful in the analysis of algorithms, including commonly encountered functions, useful formulas and appoximations, properties of logarithms, order-of-growth notation, and solutions to divide-and-conquer recurrences.

  • Definition:   logba=c log b ⁡ a = c means ab=c a b = c . We refer to b b as the base of the logarithm.

  • Special cases:   logbb=1,logb1=0 log b ⁡ b = 1 , log b ⁡ 1 = 0

  • Inverse of exponential:   blogbx=x b log b ⁡ x = x

  • Product:   logb(x×y)=logbx+logby log b ⁡ ( x × y ) = log b ⁡ x + log b ⁡ y

  • Division:   logb(x÷y)=logbx−logby log b ⁡ ( x ÷ y ) = log b ⁡ x − log b ⁡ y

Master theorem.

Let a≥1 a ≥ 1 , b≥2 b ≥ 2 , and c>0 c > 0 and suppose that T(n) T ( n ) is a function on the non-negative integers that satisfies the divide-and-conquer recurrence
T(n)=aT(n/b)+Θ(nc) T ( n ) = a T ( n / b ) + Θ ( n c )
with T(0)=0 T ( 0 ) = 0 and T(1)=Θ(1) T ( 1 ) = Θ ( 1 ) , where n/b n / b means either ⌊n/b⌋ ⌊ n / b ⌋ or either ⌈n/b⌉ ⌈ n / b ⌉ .
  • If c<logba c < log b ⁡ a , then T(n)=Θ(nlogba) T ( n ) = Θ ( n log b ⁡ a )
  • If c=logba c = log b ⁡ a , then T(n)=Θ(nclogn) T ( n ) = Θ ( n c log ⁡ n )
  • If c>logba c > log b ⁡ a , then T(n)=Θ(nc) T ( n ) = Θ ( n c )
Remark: there are many different versions of the master theorem. The Akra–Bazzi theorem is among the most powerful.
